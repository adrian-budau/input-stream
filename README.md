input-stream
============

### Usage

This crate is [on crates.io](https://crates.io/crates/input-stream) and be used
by adding `input-stream` to the dependencies in your project's `Cargo.toml`:

```toml
[dependencies]
input-stream = "0.2.0"
```

and this in your crate root:
```rust
extern crate input_stream
```
